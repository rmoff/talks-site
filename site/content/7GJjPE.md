<p>Have you ever thought that you needed to be a programmer to do stream processing and build streaming data pipelines? Think again!

Integrating Kafka with RDBMS, NoSQL, and object stores is simple with Kafka Connect, which is part of Apache Kafka. ksqlDB is the event streaming database for Apache Kafka, and makes it possible to build stream processing applications at scale, written using a familiar SQL interface.</p>
